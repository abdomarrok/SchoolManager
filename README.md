# SchoolManager

# Introduction
School Manager App is an administration tool for private schools .
This classification comes from the fact that this type (or category) of apps aims to help educational organizations in their daily routine,
by automating administrative tasks. “School manager” provides functionality from applications and class enrollment to detailed performance monitoring and financials. 
Administration management typically includes both school  and students.  
Product Overview
An open source application developed to help school managers to do their job in more efficient way with this app you could manage access to a automated platform with a given online account to create and edit student and subject that your school giving and help you to do your  finance side just by simple clicks
